# ما القيم التي ستعرض بواسطة الحلقة "for"?

قم بكتابة القيمة التي ستعرض في كل كل حلقة وقارنها بالحل.

هل ستقوم كلتا الحلقتين بعرض نفس القيم أم لا ؟

1. صيغة postfix:

    ```js
    for (let i = 0; i < 5; i++) alert( i );
    ```
2. صيغة prefix:

    ```js
    for (let i = 0; i < 5; ++i) alert( i );
    ```
