# دالة pow(x,n)

اكتب دالة `pow(x,n)` التي ترجع `x` مرفوعة لأس `n`. أو بكلمات أخرى, تضرب `x` في نفسها عدد `n` من المرات وترجع الناتج.

```js
pow(3, 2) = 3 * 3 = 9
pow(3, 3) = 3 * 3 * 3 = 27
pow(1, 100) = 1 * 1 * ...* 1 = 1
```

انشئ صفحة ويب تطلب من المستخدم قيم `x` و `n` ثم اعرض نتيجة `pow(x,n)`.
