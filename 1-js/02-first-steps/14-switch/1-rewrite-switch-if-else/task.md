# أعد كتابة "switch" باستخدام "if"

أكتب برنامج باستخدام `if..else` يطابق جملة `switch` التالية:

```js
switch (browser) {
  case 'Edge':
    alert( "لديك Edge!" );
    break;

  case 'Chrome':
  case 'Firefox':
  case 'Safari':
  case 'Opera':
    alert( 'حسنا نحن ندعم هذه المتصفحات أيضًا' );
    break;

  default:
    alert( 'نتمنى أن تكون هذه الصفحة معروضة بشكل جيد' );
}
```
